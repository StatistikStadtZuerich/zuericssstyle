# sszOgdDownload

function to generate an ssz-themed ogd download button

## Usage

``` r
sszOgdDownload(outputId, href, label, image = NULL)
```

## Arguments

- outputId:

  outputId parameter

- href:

  link to be opened

- label:

  label parameter (i.e. text to be shown)

- image:

  optional image/icon in the form of a html-i Tag, default NULL

## Value

tags\$a
